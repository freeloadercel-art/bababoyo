HOLODOMOR

Spatial Analysis of Stalin’s Famine-Genocide in Ukraine, 1932–1934
INTERACTIVE PROJECT

Introduction & Approach

This project presents a spatial analysis of the Holodomor, using interactive maps, charts, and visualizations to examine how famine and mortality patterns varied across Ukraine from 1931–1934. The aim is to show patterns of mortality, grain requisitions, blacklists, and survivor testimonies in a way similar to a scientific study: hypotheses are stated, visual evidence is presented, and analysis is provided for each dataset.
